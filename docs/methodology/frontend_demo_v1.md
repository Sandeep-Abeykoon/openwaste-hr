# Simple Frontend Demo Page v1

## Purpose

This stage creates a simple frontend demo page for OpenWaste-HR.

The page allows a user to enter an image path and sample ID, send the request to the backend inference endpoint, and view the final OpenWaste-HR decision.

## Frontend Files

| File                | Purpose                               |
| ------------------- | ------------------------------------- |
| frontend/index.html | page structure                        |
| frontend/styles.css | visual styling                        |
| frontend/app.js     | backend API call and result rendering |

## Backend Endpoint Used

The frontend calls:

```text
POST http://127.0.0.1:8000/api/inference/predict
```

## Request Fields

| Field      | Meaning                            |
| ---------- | ---------------------------------- |
| image_path | project-relative path to the image |
| sample_id  | optional sample identifier         |
| request_id | generated by the frontend          |

## Displayed Output

The frontend displays:

| Output              | Meaning                                    |
| ------------------- | ------------------------------------------ |
| status              | backend response status                    |
| predicted label     | model fine-label prediction                |
| confidence          | model confidence                           |
| decision type       | fine_label, coarse_label, or manual_review |
| final label         | final OpenWaste-HR result                  |
| decision reason     | explanation of the decision                |
| class probabilities | probability for each known fine label      |

## Research Meaning

This stage turns the backend endpoint into a visible project demonstration.

The system can now be shown as a prototype workflow: a user enters an image path, the backend runs inference, and the frontend displays a safe hierarchical decision.
